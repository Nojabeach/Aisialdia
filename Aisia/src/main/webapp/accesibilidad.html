Accesibilidad

Estamos comprometidos a garantizar que nuestro sitio web sea accesible para todas las personas, incluidas aquellas con discapacidades.

Accesibilidad del sitio web

Este sitio web ha sido diseñado con principios de accesibilidad en mente, incluyendo el uso de marcado semántico, etiquetas adecuadas y un diseño adaptable.

Comunicación

Si encuentra alguna barrera de accesibilidad en nuestro sitio web o tiene alguna sugerencia para mejorarlo, no dude en ponerse en contacto con nosotros.

Compatibilidad con tecnologías de asistencia

Nuestro sitio web es compatible con tecnologías de asistencia como lectores de pantalla y navegadores de voz, para garantizar una experiencia de usuario óptima para todos.
